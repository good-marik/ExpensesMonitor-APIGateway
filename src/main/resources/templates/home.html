This web application is designed to help you to track and monitor your expenses, positively impacting your savings strategies.

It follows a microservices architecture, consisting of a gateway, a REST API server, and a database (scheme).

Technologies used: Java, Spring Boot, Spring Security, Spring Data JPA, PostgreSQL, and Thymeleaf.

The project is written by Marat Khusniyarov for his portfolio.

The original code can be found at GitHub.
